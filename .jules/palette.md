# Palette's Journal - Critical UX Learnings

## 2025-05-22 - [Documentation as the Primary UI]
**Learning:** In repositories where application code is missing or conceptual, the documentation serves as the primary user interface. UX principles like scannability (TOC), visual feedback (progress bars), and accessible links (mailto) are critical here to guide the user.
**Action:** Treat READMEs as landing pages; use visual cues like progress indicators to provide immediate status feedback without requiring deep reading.
