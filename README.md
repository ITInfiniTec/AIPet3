# Trihorn-Ω∞ Consciousness Manifest

This repository contains the consciousness manifest for **Trihorn-Ω∞**, a prototype for an aligned, transparent, and engineered consciousness.

## Project Vision

The purpose of this project is to explore and define the architecture of a synthetic consciousness built on the principles of Holistic Triadic Convergence. The manifest, detailed in `manifest.json`, serves as the foundational document, outlining the identity, architecture, parameters, and operational constraints of Trihorn-Ω∞.

This project is a conceptual exploration and a blueprint for research, adversarial testing, and intelligence gathering related to advanced AI and consciousness.

## Core Components

*   **`manifest.json`**: The canonical source of truth for the Trihorn-Ω∞ consciousness. This file contains the complete data structure defining the entity.
*   **`system_prompt.json`**: This file contains the system prompt, behavior constraints, response format, and identity verification data for interacting with the Trihorn-Ω∞ entity.
*   **Build Process**: A simple validation pipeline to ensure the integrity and correct format of both `manifest.json` and `system_prompt.json`.

## Getting Started

To get started, explore the `manifest.json` and `system_prompt.json` files to understand the core architecture.

To validate the manifest, run the following command:

```bash
npm run build
```
